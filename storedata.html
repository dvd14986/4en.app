Store data into the blockchain

STEP 1: Choose destination blockchain!

Metamask: Custom RPC not required. Select Main Ethreum Network from Metamask before. Webpage could refresh after network set on Metamask.
Metamask: https://etc-geth.0xinfra.com/ Select Custom RPC and insert ETC RPC before go to the next step. Webpage could refresh after network set on Metamask.
Metamask: https://clo-geth.0xinfra.com/ Select Custom RPC and insert CLO RPC before go to the next step. Webpage could refresh after network set on Metamask.

STEP 2: Write your text or upload your file!

Upload your file!

You can upload all type of file but with some limitation:

  • Size cannot exceed 5Kbyte.
  • File cannot contain null(\u0000) char.
  • File must be UTF-8 encodable.
Unrecognized type of file, or file that don't fills the above requirements will not be accepted.

STEP 3: Store it!

Destination blockchain:
Click on Store it! button and confirm the operation from Metamask window. If it don't opens automatically, click on icon in the browser bar.
Remember to increase the default Metamask gas price to speed up the validation from the network!
Transaction could take some time to be processed from the nodes, then wait Metamask confirmation to check if the storage success from Get data page.
Save your transaction hash to recover your file later!
