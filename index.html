Store your text and little documents on blockchain. Easy, cheap and fast!

About us

4en.app is a simple service that allow anyone to store data on the most popular blockchains with few clicks! You can use it to store notes and documents. Blockchains technologies give security and assure permanent storage with no chance to edit or delete your content!

How it works

This application is powered by several smart contracts deployed each on it own blockchain. When process is started, webapp collect all data provided by the users and push it to the smart contract related to the selected blockchain.

The smart contract take the data and insert it into single transaction, then request the payment via Metamask. After payment confirmation via Metamask, transaction is started and your data is finally push to the blockchain. To view your data, simply use transaction Hash.
