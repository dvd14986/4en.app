Get Started

Summary

4en.app is a platform that allow anyone to store some message, data or file into Ethereum and Ethereum like blockchain network.
The process is very easy and involve only 3 steps:
  1. Choose Destination blockchain
  2. Upload a file or write a message
  3. Store it!
Only one page to make the process! After storing, the data could be recovered from Get Data page, by simply insert the hash of the storing transaction.
System will search on all supported blockchains and show the result. If it find a file you can download a copy of it!

Setup the enviroment and select the destination blockchain

First step is choose the destination blockchain.
Actually Ethereum and Ellaism have full Metamask support and work properly with 4en.app.
Ethereum Classic and Callisto Network is actually fully supported by 4en.app, but not from Metamask. For this reason it will be not selectable, at moment, as destination blockchain, until Metamask don't extend support to that network. Metamask developer still working on this feature.
Store data has a cost because it burn some gas to run the smartcontract in the destination network. For this reason you need to install Metamask for use this service.
Metamask provide a wallet and 4en.app take funds from it. It also support other network than Ethereum, simply adding new RPC custom node. The RPC node to add is reported after blockchain selection box in the Store Data page. Add it to metamask before go on into the store process. When add or switch from metamask network, a page refresh could occur and maybe you have to select again the destination blockchain, according to Metamask selection.
Visit metamask.io to install the browser extension and to learn how it works.

How to get coins

Click on the coin name to check exchange listing it.
  • Ethereum(ETH)
  • Ellaism(ELLA)
  • Ethereum Classic(ETC)
  • Callisto Network(CLO)

Storing

Storing function support a wide range of file, but, due to some enviroment limitation, some file could not be accepted.
An alert message inform of unacceptable file.
Main limits on files upload are:
  • File size cannot exceed 5Kbyte.
  • File cannot contain null(\u0000) char.
  • File must be UTF-8 encodable.
If you want to store only text, you can insert in the textbox up to 5000 char.
When ready, click on Validate Button!
If data is ok, last step will be enabled.
Clicking on Store it! button, start the process. Metamask will ask you confimation for the transaction and after few second transaction hash appear, with an explorer link to see the transaction. Depending on network congestion, settings of gas price and network global hashrate, transction could take some time to be verified and inserted into a block. Metamask will inform you when it is done.
From this moment the transactio content could be checked and view from Get Data.

View and download your data

Insert the Transaction hash received during stor process and click on Search button.
System will search on each supported blockchain for that has and will show the result with some additional data. If a 4en record will found on the transaction data, a link will displayed for decode the Tx data content. In case of file, a download button allow user to download it.
All GET process is accomplished without any smart contract and is totally free!
